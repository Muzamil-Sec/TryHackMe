Linux Fundamentals Part 1
Overview

This room provided a strong introduction to the Linux operating system and the foundational concepts required to work comfortably in a Linux environment.

Key Concepts Learned

Understanding the Linux command-line interface and navigating the terminal effectively.

Using basic file system commands such as listing directories, creating files, copying, moving, and deleting resources through the shell.

Learning file paths, relative versus absolute paths, and the structure of a Linux directory hierarchy.

Gaining familiarity with viewing and editing file content using standard commands like cat, head, tail, and nano.

Introduction to user roles and permissions within a Linux system.

Skills Developed

Confident use of terminal navigation and command execution.

Handling and organizing files directly from the shell.

Interpreting directory structures and understanding where system files are stored.
