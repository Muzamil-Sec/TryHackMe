Linux Fundamentals Part 2
Overview

This room built on foundational knowledge and expanded into deeper file management, permissions, and useful utilities that are essential for system operations.

Key Concepts Learned

Advanced permissions management including read, write, and execute rights for users, groups, and others.

Using chmod, chown, and groups to control access and file security.

Working with processes: listing, terminating, and understanding process management with commands such as ps, kill, and top.

Introduction to data searching and filtering using grep and wildcards.

Basic understanding of package management for installing and maintaining software.

Skills Developed

Efficient troubleshooting and system control through process management.

Applying strict security rules to files and directories.

Searching through large logs or directories with precision.
