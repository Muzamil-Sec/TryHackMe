Linux Fundamentals Part 3
Overview

This room introduced more operational and administrative commands that are critical for real-world Linux usage, especially in security and system engineering roles.

Key Concepts Learned

Networking basics including hostname, IP addressing, and managing network interfaces using commands like ip and netstat.

Log analysis and understanding where system logs are stored for monitoring and auditing.

Service and system management using systemctl to start, stop, and configure services.

Introduction to archives and compression with tar, gzip, and related utilities.

Understanding environment variables, user profiles, and bash configuration files.

Skills Developed

Conducting network diagnostics and understanding basic connectivity issues.

Maintaining and reviewing system logs to identify potential security or operational concerns.

Managing services that keep a Linux system running smoothly.

Handling backup and data transfer tasks through compression and archiving.
