Windows Fundamentals 2
Overview

This room focused on system security features, user access control, and deeper OS components, helping build strong defensive understanding for practical Windows management.

Key Concepts Learned

Windows permissions, NTFS file system security, and understanding Access Control Lists (ACLs).

How Windows handles authentication and authorization for local users and groups.

BitLocker data protection and how encryption ensures confidentiality of stored data.

Overview of key built-in administrative utilities such as Local Security Policy and Event Viewer.

System logs and how they assist in monitoring security-related events and troubleshooting.

Skills Developed

Applying and editing file permissions to secure resources.

Managing local user accounts and enforcing security policies.

Conducting basic incident investigation using logs and security settings.
