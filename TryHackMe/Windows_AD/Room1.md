Windows Fundamentals 1
Overview

This room introduced the core structure, functionality, and key components of the Windows operating system environment, essential for anyone entering cybersecurity and system administration.

Key Concepts Learned

Understanding the Windows graphical user interface and how it differs from Linux-based systems.

Introduction to the Windows file system architecture including drives, file paths, and directory structures.

Familiarity with built-in Windows tools such as Control Panel, Settings, File Explorer, and Task Manager.

Basic concepts of user accounts, system roles, and privileges.

System navigation using both GUI and Command Prompt for executing essential tasks.

Skills Developed

Efficiently locating system utilities and configuration tools.

Navigating and managing files and folders in Windows.

Understanding the foundational elements of Windows security and administration.
