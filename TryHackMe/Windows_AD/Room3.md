Windows Fundamentals 3
Overview

The third room explored essential Windows networking functionality, core management tools, and processes that keep the operating system running securely and reliably.

Key Concepts Learned

Windows networking stack including IP configuration and tools like ipconfig, ping, and netstat.

Service and process management with tools such as Services app, Task Manager, and PowerShell.

Introduction to Windows Registry as a critical configuration database and how settings are stored.

Understanding startup programs, scheduled tasks, and background services.

File shares, network discovery, and how Windows devices communicate within a local network.

Skills Developed

Diagnosing network issues and connectivity failures.

Configuring and managing system services to ensure system stability.

Reading and modifying registry values responsibly for troubleshooting and optimization.

Securing and accessing shared network resources within a Windows environment.
