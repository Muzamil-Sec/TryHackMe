Active Directory Basics
Overview

This room provided foundational knowledge of Active Directory, the centralized identity and access management system used widely in enterprise environments.

Key Concepts Learned

Understanding what Active Directory is and why organizations rely on Domain Controllers for centralized control.

Core AD components: Users, Groups, Organizational Units (OUs), Computers, Policies.

The structure and function of a Windows Domain and how authentication works using Kerberos.

Group Policy Objects (GPOs) for enforcing organization-wide security and configuration rules.

Domain-based access authorization and how permissions are applied to network resources.
